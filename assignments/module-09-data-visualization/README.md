# assignment9-visualization-r
# Assignment 9 - Visualization in R

This repository contains my work for Assignment #9: Visualization in R – Base Graphics, Lattice, and ggplot2.

## Files
- `assignment9.R` - R script containing all code for the assignment

## Dataset
I used the built-in `iris` dataset from the `datasets` package in R.

## Visualizations Included
- Base R scatter plot
- Base R histogram
- Lattice conditioned scatter plot
- Lattice boxplot
- ggplot2 scatter plot with smoothing
- ggplot2 faceted histogram

## Purpose
The goal of this assignment was to compare three R visualization systems:
1. Base graphics
2. Lattice
3. ggplot2
