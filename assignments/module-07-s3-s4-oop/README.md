# Module 07 – S3 vs S4 Object-Oriented Programming in R

## Overview
Explored object-oriented programming in R by comparing S3 and S4 systems using custom objects and the mtcars dataset.

## Key Concepts
- S3 classes and method dispatch
- S4 class definitions and slots
- Generic functions (e.g., `summary()`)
- Differences between S3 and S4 systems
