# Assignments – Advanced R Programming

This folder contains all coursework completed for the **Advanced R Programming** course. Each module focuses on a specific concept in R, including data structures, matrix operations, object-oriented programming, data visualization, and reproducible workflows.

Each assignment includes:
- An R script (`.R`) with the implementation
- A `README.md` explaining the approach and results
- Supporting files (datasets, plots, or outputs when applicable)
- A link to the corresponding blog post
