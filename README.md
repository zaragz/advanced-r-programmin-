# Advanced R Programming

This repository contains coursework and projects completed for the graduate-level course **Advanced R Programming**.

The course focuses on advanced scripting in R, data structures, object-oriented programming (S3/S4), data analysis, and reproducible workflows. The repository also includes a final project involving the development of an R package.

---

## Repository Structure

### Assignments
- Module 03: Dataframe Operations
- Module 03: Determinant and Inverse Matrices
- Module 07: S3 and S4 Object-Oriented Programming
- Module 08: Assignment
- Assignment 09: Data Visualization in R
- Additional R programming exercises

### Analysis Projects
- **Hospital Blood Pressure Analysis**  
  Exploratory and statistical analysis of hospital blood pressure data.

- **Matrix Operations in R**  
  Implementation and analysis of matrix computations in R.

### Final Project
- R package development project (in progress / completed)
