# Module 12 – Introduction to R Markdown

## Overview
Created an R Markdown document that combines writing, code, and output in a single file. This assignment focused on learning Markdown structure, LaTeX math formatting, and reproducible analysis.

## Key Concepts
- Writing narrative text using Markdown
- Embedding R code chunks
- Using LaTeX for inline and display math
- Generating plots with ggplot2
- Knitting documents to HTML
