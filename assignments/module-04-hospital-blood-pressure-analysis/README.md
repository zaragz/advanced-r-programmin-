# Module 04 – Hospital Blood Pressure Analysis

## Overview
Analyzed hospital patient data in R, including blood pressure, visit frequency, doctor assessments, and final treatment decisions.

## Key Concepts
- Data frame creation
- Recoding categorical variables
- Boxplots and histograms
- Summary statistics with `summary()` and `tapply()`

## Files
- `module-04-hospital-blood-pressure-analysis.R` — Main analysis script
- `module-04-boxplot.png` — Single boxplot of blood pressure
- `module-04-boxplots.png` — Side-by-side boxplots by doctor ratings and final decision
- `module-04-histogram.png` — Histogram of blood pressure values
