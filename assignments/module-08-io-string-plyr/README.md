# Module 08 – Input/Output, String Manipulation, and plyr in R

## Overview
Worked with data import, string filtering, and grouped summaries in R using the plyr package.

## Key Concepts
- Importing data with `read.table()`
- Grouped calculations using `ddply()`
- String filtering with `grepl()`
- Exporting data to CSV files
